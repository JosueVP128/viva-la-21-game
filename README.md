## Description
This project is a simple, beginner-friendly Blackjack simulation game. This game features the basic concept of Blackjack's hitting and standing, alongside other features such as a dealer who reacts to the result of a round, a credit system, and a double-down mechanic. This game also features music and sound effects.

The different files in the repository contain assets for the game. Assets include the cards, the dealer's reactions, the title screen, the music, and the sound effects. The cards are essential to Blackjack, as Blackjack is a card game. The other assets help make the game more dynamic and not just a boring Blackjack game.

When designing this game, I wanted it to be a beginner-friendly version of Blackjack. This is why some of the other more complex mechanics of Blackjack, such as splitting and insurance have been omitted from the game. However, the game still functions as Blackjack, with hitting, standing, and doubling down being identical to how they work in real Blackjack. Also, just like real Blackjack, if the player gets a blackjack, they are paid 1.5 times the bet.

Future areas for improvement would be improving the visuals and animations. While the visuals are functional, they could look somewhat better. As for animations, it would be nice to see the cards slide in instead of just popping into the scene.
