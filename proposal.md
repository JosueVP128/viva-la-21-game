# Viva la 21 (Blackjack Game)

## Repository
<https://github.com/JosueVP128/viva-la-21-game.git>

## Description
This project is a simulation of the card game Blackjack in PyGame. A betting system and dealer character will be featured to make the game more unique.

## Features
- Feature 1
	- Players will be able to play Blackjack thanks to PyGame and the random module.
- Feature 2
	- The dealer will react to the player's results using if statements and custom sprites.
- Feature N 
	- A betting system will be implemented using if statements.

## Challenges
- Research how to make the cards be animated.
- Research how to differentiate 1 and 11 when an Ace is played.
- Research how to make sure the dealer uses the right reactions during gameplay.

## Outcomes
Ideal Outcome:
- The ideal outcome is to have a perfectly replicated version of Blackjack with a betting system and dealer who reacts perfectly to the outcomes.

Minimal Viable Outcome:
- The minimal viable outcome is to have a simple and basic Blackjack simulator. This means that the betting system and expressive dealer will not be featured in the game.
